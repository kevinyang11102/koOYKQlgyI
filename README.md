# 社团管理系统 java1129

## 项目概述

社团管理系统是一款基于B/S架构的Web应用程序，采用MVC模式设计，实现前后端分离。该系统致力于为各类社团提供全面的管理解决方案。

## 技术栈

- **后端**：Java、SpringBoot、Mybatis
- **前端**：Vue
- **数据库**：Mysql
- **开发工具**：IDEA、JDK1.8、Maven

## 系统角色

系统定义了以下角色，并分配了不同的权限和功能：

- **用户**：普通登录用户，可进行基础操作。
- **超级管理员**：拥有对系统的全面管理权限。
- **社团管理员**：负责管理本社团相关事务。
- **学生用户**：参与社团活动，管理个人信息。

### 功能模块

- **用户模块**
  - 登录用户
  - 注册用户
  - 用户信息管理

- **超级管理员**
  - 所有社团管理

- **社团管理员**
  - 本社团信息管理
  - 社管活动管理
  - 入团申请
  - 费用管理
  - 社团活动通知

- **学生用户**
  - 社团信息管理
  - 社团活动管理
  - 入团社区
  - 费用管理
